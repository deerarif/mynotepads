
# My Notepads

This is my fist project using MERN Stack for building Notepads web/apps

## Authors

- [@deerarif](https://www.github.com/deerarif)


## Setups

Clone the project

```bash
  git clone https://www.github.com/deerarif/mynotepads.git
```

Go to the project directory

```bash
  cd mynotepads
```

Then just run

```bash
  docker-compose up
```
